1. A solução deve ser disponibilizada em Cloud ou On-premise?
 - Em cloud, pois os custos de uma infra estariam terceirizados, atualizações de hardware softwares. Plataformas de Cloud-infra também possuem diversos recursos
 para propociornar a escalabilidade e também a segurança de uma aplicação.

2. Considerando a participação de 4 pessoas técnicas para o desenvolvimento dessa solução, responda qual ferramenta acreditas ser apropriada para o controle de tarefas da equipe, a ferramenta é ágil?
 - Pensando numa esteira de desenvolvimento, acredito que o Scrumban (Scrum + kanban) permite com que a equipe visualize o que todos estão trabalhando e a prioridade
 de começar a terminar em vez de puxar novas tarefas ajuda no ganho de entrega de valor para o cliente.
 Uma esteira poe exemplo: backlog -> refinamento -> to do -> in progress -> code review -> qa(dev) -> qa(staging) -> deploy prod -> monitoring -> done
 é um fluxo de trabalho que vejo cobrir as pricipais etapas de um desenvolvimento de app.

 O jira pode ser uma ferramento que ajuda criarmos esse ambiente de iteração e a esteira.
